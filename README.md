# robosys2022
このプログラムはロボットシステム学2022年度の講義で作成されました。

## plusコマンド
![test](https://github.com/ogi-y/robosys2022/actions/workflows/test.yml/badge.svg)

標準入力から1行づつ読み込み、数字であれば記号に基づき四則演算をし、文字列であればつなげて出力します。
